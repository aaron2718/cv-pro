### Usage
Insert your own contents.
Change values of the `\newcommand` definition to adjust spacing in one place for the whole document.

### Requirements
Make sure you have the following packages installed (debian):
* `texlive-fonts-extra`
* `texlive-lang-german`
* `texlive-latex-extra`

### Note !
Stay with the empty line after the `\CAPTION` command to have proper spacing.
