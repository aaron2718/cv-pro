# cv-pro
Latex template for a professional CV

It could be that you have to install some additional Latex packages like `texlive-fonts-extra`, `texlive-lang-german` and `texlive-latex-extra` (debian).
The PDF file shows the design of the document.

In the TEX file are some lines with the comment `% change values if needed`.
There you can adapt the spacing of your CV.
It is recommended to stick with the empty line after the CAPTIONs to have a proper spacing underneath them.
